## 🚀 Getting started

### Environment configuration

[**React Native - Setting up the development environment**](https://reactnative.dev/docs/environment-setup)

NodeJS: +16.0

### Create .env
Ver archivo .env.example

```
API_URL=https://graphql-weather-api.herokuapp.com/
```

## 🔸 Scripts

**Ejecutar en Android**

`npm run android`

**Tests**

`npm run test`

`npm run test:watch`
